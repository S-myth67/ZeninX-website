# Image Directory

## Where to place your images:

### Dark Theme Images (Current)
Place your 8 dark theme images here:
- **7 Android format images**: Name them `app-dark-1.png`, `app-dark-2.png`, etc. (or keep your original names)
- **1 big image with 3 sections**: Name it `app-hero-dark.png` (or keep your original name)

### Light Theme Images (Future)
When you add light theme images later, place them here:
- **7 Android format images**: Name them `app-light-1.png`, `app-light-2.png`, etc.
- **1 big image with 3 sections**: Name it `app-hero-light.png`

## Image Structure:
```
public/images/
├── app-dark-1.png (or your original name)
├── app-dark-2.png
├── app-dark-3.png
├── app-dark-4.png
├── app-dark-5.png
├── app-dark-6.png
├── app-dark-7.png
├── app-hero-dark.png (the big image with 3 sections)
└── (light theme images will go here later)
```

**Note**: The component will automatically detect dark/light mode and show the appropriate images.
